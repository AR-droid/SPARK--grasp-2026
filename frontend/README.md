# SPARK AI Frontend

This is the React frontend for the SPARK AI Asset Integrity Management platform.

## Prerequisites

-   **Node.js**: You must have Node.js installed (LTS version recommended).
    -   Download here: [https://nodejs.org/](https://nodejs.org/)
    -   Verify installation: `node -v` and `npm -v`

## Getting Started

1.  **Open Terminal**: Open your terminal or command prompt in this directory (`frontend/`).
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
4.  **View Application**:
    -   Open your browser vsit the URL shown in the terminal (usually `http://localhost:5173`).

## Project Structure

-   `src/pages`: Core application pages (Landing, Dashboard, etc.)
-   `src/components`: Reusable UI components
-   `src/App.jsx`: Main routing configuration
-   `src/index.css`: Global styles aka "Dusty" design system
